# Mole 
Mole破-解 for mac, no official subscription required


## 以下是原Mole软件介绍：
**🐹 视觉化清理、卸载、优化与实时监控你的 Mac**

Mole for Mac 是 Mole 命令行工具的原生图形界面版本，提供更直观的操作体验。结合 CleanMyMac、AppCleaner、DaisyDisk 和 iStat Menus 的功能于一体，帮助你轻松释放磁盘空间、管理应用和监控系统健康。

**CLI 版本免费开源**，Mac 应用提供更丰富的可视化功能。

---

## ✨ 主要特性

- **智能清理**：可视化预览并清理缓存、日志、残留文件，一键释放 GB 级空间
- **强大卸载器**：彻底移除应用及其所有隐藏残留（Launch Agents、Preferences 等）
- **磁盘地图**：直观的可视化磁盘占用分析，支持大文件查找和删除
- **实时监控**：菜单栏 HUD + 系统状态仪表盘（CPU、GPU、内存、磁盘、网络、温度）
- **应用更新**：一键检查并更新已安装应用
- **维护工具**：系统优化、重建缓存、重置服务等
- **安全优先**：干运行预览、受保护路径、智能确认机制

---

## 📥 下载与安装

1. 打开 DMG 文件，将 **Mole** 拖拽到 **Applications** 文件夹
2. 首次启动时，在 **系统设置 → 隐私与安全性** 中允许 Mole（可能需要授予完全磁盘访问权限以实现最佳效果）

**推荐**：授予「完全磁盘访问权限」以获得完整清理和监控功能。

---

## 🚀 使用方式

- 启动 Mole 后，通过左侧菜单或顶部栏快速访问不同功能
- **Clean**：可视化清理预览 + 一键执行
- **Uninstall**：选择应用并彻底移除
- **Analyze**：磁盘占用可视化探索
- **Status**：实时系统仪表盘（支持菜单栏常驻）
- **Maintenance**：系统优化工具

更多详细命令和高级用法请参考 [CLI 版本文档](https://github.com/tw93/Mole)。
